# Pewlett Hackard Employee Postgres Database


## Tools/Packages
- SQL
- PostgreSQL
- pgAdmin

## How to run
- Download the respository
- Open pgAdmin (for PostgreSQL) and create database named **Employees**
- Populate the database tables in the following order:
  - titles
  - departments
  - employees
  - salaries
  - dept_manager
  - dept_emp
